# じゃんけんゲーム 開発タスク

## 目標
Pythonでじゃんけんゲームを作る。プレイヤーがCPUと対戦できるCLIアプリ。

## タスク一覧

### 1. 基本機能
- [ ] プレイヤーの手入力（グー / チョキ / パー）
- [ ] CPUのランダムな手を生成
- [ ] 勝敗判定ロジック（勝ち / 負け / あいこ）
- [ ] 結果をターミナルに表示

### 2. ゲームループ
- [ ] 繰り返しプレイできるようにする（「もう一度？」の選択肢）
- [ ] ゲーム終了の処理

### 3. スコア管理
- [ ] 勝ち / 負け / あいこ の累計を記録
- [ ] ゲーム終了時にスコアを表示

### 4. 入力バリデーション
- [ ] 無効な入力に対してエラーメッセージを表示し再入力を促す

### 5. 仕上げ
- [ ] コードの整理・リファクタリング
- [ ] 動作確認・テスト

## ファイル構成（予定）
- `janken.py` — メインのゲームコード
